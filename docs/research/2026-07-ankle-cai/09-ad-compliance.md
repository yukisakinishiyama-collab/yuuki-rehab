# 成果物9：医療広告チェック済み表現集

担当：Marketing AI ／ 作成日：2026-07-12 ／ 状態：CEO 承認待ち
適用範囲：Instagram・GBP・note・ホームページ・院内掲示・チラシ等、すべての対外発信

## 1. 法的前提（整骨院＝柔道整復師の広告規制）

- **柔道整復師法 第24条**：広告できる事項は限定列挙（施術者の氏名・住所、施術所の名称・電話・所在地、施術日・施術時間 等）。施術内容・技能・経歴等の広告は原則不可
- **医療広告ガイドライン（厚労省）**の趣旨：虚偽・誇大・比較優良・患者体験談による誤認・ビフォーアフターの不適切使用の禁止。ウェブサイト・SNS も実質的に広告として扱われる流れにあり、安全側で同基準を適用する
- **景品表示法**：優良誤認（実際より著しく優良と誤認させる表示）の禁止
- 当院方針（`docs/ai-org/clinic-profile.md`）：法の解釈で攻めず、**常に安全側**で統一

## 2. 禁止表現 → 代替表現 対応表

| ✕ 禁止 | ○ チェック済み代替表現 |
|---|---|
| 治る／完治します | 「回復を目指して段階的にサポートします」 |
| 根本改善 | 「再発予防を目的としたトレーニングを行います」 |
| 必ず改善／絶対に良くなる | 「〜が有効だったという研究報告があります（効果には個人差があります）」 |
| 地域No.1／下関で一番 | 「学生アスリート・ダンサーの来院が多い院です」（事実の範囲で） |
| 病院で治らなかった症状が治る | 「医療機関と連携しながらリハビリを進めます」 |
| 捻挫ぐせを治します | 「繰り返す捻挫の背景にある機能低下を評価し、再発予防に取り組みます」 |
| 最新の／特別な施術 | 「研究知見を参考にしたプログラム」 |
| ○○選手も通院（体験談・推薦） | 使用しない |
| 施術前後の写真で効果を暗示 | トレーニング風景・院内写真のみ使用 |

## 3. 数値引用の統一ルール

1. 引用する数値は**公刊論文由来のみ**。院独自の実績数値（改善率・満足度等）は使用しない
2. 数値には必ず出典条件を付す：「※海外の研究報告に基づく数値です」
3. 「約4割が CAI に」は必ず「**分類されたという報告があります**」の形で使う（断定形・恐怖訴求の見出し単独使用は不可）
4. 効果に言及する場合は「効果には個人差があります」を近接配置する

## 4. 必須併記事項

対外発信物には以下のいずれかを文脈に応じて含める：

- 骨折等が疑われる場合は医療機関の受診を案内する旨
- 医師・医療機関と連携して進める旨
- 効果には個人差がある旨（効果に言及した場合）

## 5. 公開前チェックリスト（全発信物に適用）

- [ ] 「治る・完治・根本・必ず・No.1」等の禁止語が含まれていない
- [ ] 医師・病院・他院への否定的言及がない
- [ ] 数値に出典注記があり、断定形になっていない

- [ ] 柔道整復師（施術管理者）の最終確認を得た
