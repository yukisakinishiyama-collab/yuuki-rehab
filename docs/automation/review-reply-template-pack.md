# 口コミ返信テンプレート集 v1.0（フロー #002 用）

作成：Marketing AI（Legal AI 確認済み）／ 作成日：2026-07-12 ／ 状態：**CEO_INPUT_REQUIRED（テンプレート承認待ち）**
運用原則（CEO 決定・2026-07-12）：**完全自動返信は禁止**。肯定＝AI下書き→CEO一括承認→返信、中立＝AI下書き→CEO確認→返信、ネガティブ・医療内容・金銭・保険・事故・クレーム＝**自動返信禁止・L3手動対応**。

## 共通ルール（全テンプレート適用）

- 構成：感謝 → 一般的な方針（任意）→ 今後の丁寧な対応、のみ。80〜200字
- 口コミ本文の症状・個別事情を返信で**繰り返さない**
- 来院事実を院側から**過度に確認しない**（「ご来院ありがとうございました」は投稿者自身が来院に言及している場合のみ可。迷ったら「ご投稿ありがとうございます」）
- 効果への同調（「良くなって何よりです」等の施術効果の追認）をしない
- 診断的応答・医学的アドバイス・営業的勧誘をしない

※要望の内容（待ち時間・料金・対応等）を返信内で個別に釈明・反論しない。改善姿勢のみ示す。

## T-8 ネガティブ口コミ時の初動メモ（返信テンプレートではない・L3手引き）

1. **返信しない**（下書きも投稿しない）。まず CEO へ即時共有（CEO_INPUT_REQUIRED・緊急）
2. 事実確認：来院記録・当日の状況をスタッフから聴取（口コミ投稿者の特定を目的としない。事実経過の把握のみ）
3. Legal AI に分類を依頼：①正当な苦情 ②事実誤認 ③ポリシー違反疑い（誹謗中傷・無関係・なりすまし）
4. ①は改善姿勢の返信文を CEO＋Legal 両承認後に投稿。②は反論せず「事実確認のうえ直接ご連絡いただきたい」旨のみ。③は返信より Google への報告を優先検討
5. 返信は1回まで。再燃時は追加返信せず L3 継続（炎上対応手順へ）
6. 全経過を監査ログに記録

## T-9 返信してはいけない例（禁止パターン集)

| 禁止例 | 理由 |
|---|---|
| 「捻挫の経過が良くなられたようで何よりです」 | 症状の繰り返し＋効果の追認（守秘・広告の両面で不可） |
| 「〇月〇日にご来院いただいた際は…」 | 来院事実・日時の確認（個人情報） |
| 「それは当院の施術の効果です」 | 効果の主張（広告規制） |
| 「その点は誤解です。実際には…」 | 口コミ上での反論・事実関係の争い |
| 「次回ご来院時に割引いたします」 | 対価提供（Google ポリシー・ステマ規制） |
| 「お知り合いにもぜひご紹介ください」 | 営業的勧誘 |
| 「病院より当院の方が…」 | 比較優良表現 |
| 「保険が使えますのでぜひ」 | 保険適用の誤認誘導 |
