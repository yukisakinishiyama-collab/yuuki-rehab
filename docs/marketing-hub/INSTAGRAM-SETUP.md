# Instagram自動投稿の接続手順

対象: yu.ki__seikotsuin ／ 所要30〜45分 ／ 無料
これを設定すると「AI生成→承認→予約→時刻が来たら画像付きでInstagramへ自動投稿」が完成する。

## 前提条件（先に確認）

1. **Instagramがプロアカウント**（ビジネスまたはクリエイター）であること
   - Instagramアプリ → プロフィール → 設定 → アカウントの種類 → プロアカウントに切り替え（無料）
2. スマホのInstagramアプリで切り替え操作をするため、スマホが必要

## 手順

### Step 1: Meta開発者アカウントとアプリ作成

1. https://developers.facebook.com/ を開き、Facebookアカウント（個人）でログイン
2. 「マイアプリ」→「アプリを作成」
3. ユースケース: 「その他」→ アプリタイプ: 「ビジネス」
4. アプリ名: `yuuki-marketing` 等（何でもよい）

### Step 2: Instagram APIをセットアップ

1. アプリのダッシュボード → 製品を追加 → 「**Instagram**」の「設定」
2. 「**Instagram APIのセットアップ（Instagramログイン）**」を選択
3. 「Instagramアカウントを追加」→ yu.ki__seikotsuin でログインして連携

### Step 3: アクセストークンとユーザーIDの取得

1. 同じ画面の「アクセストークン生成」セクションで、連携したアカウントの「トークンを生成」
2. 表示された**長期アクセストークン**をコピー → Vercelの `INSTAGRAM_ACCESS_TOKEN` に設定
3. 同じ画面に表示される**InstagramユーザーID**（数字）をコピー → `INSTAGRAM_USER_ID` に設定
4. 再デプロイ（コード変更をpush、またはVercelのRedeploy）

### Step 4: 動作確認

1. マーケティングハブ →「投稿ジョブ」→ API接続確認で Instagram が「接続済み」になる
2. 「投稿を作る」でテスト投稿を1本生成 → 承認 → 数分後の時刻で予約
3. 時刻が来たら自動実行（GitHub Actions 15分間隔）→ Instagramに画像付きで公開される
4. 不安な場合は最初の1本を深夜に予約し、翌朝確認してから運用開始

## トークンの有効期限

- 生成される長期トークンは約60日有効。期限が近づくと投稿が「公開失敗」になるので、
  その場合はStep 3で再生成→Vercel差し替え（将来の自動リフレッシュは改善候補）

## 補足

- 自動投稿は**承認済み→予約済み**のジョブだけが対象。勝手に投稿されることはない
- 画像は投稿テーマからネイビーブランドのテンプレート画像を自動生成
  （院内写真を使いたい投稿は、手動投稿モード＝本文コピー運用も引き続き使える）
- カルーセル・リールの自動投稿は段階対応（現状は手動対応待ちになる）
