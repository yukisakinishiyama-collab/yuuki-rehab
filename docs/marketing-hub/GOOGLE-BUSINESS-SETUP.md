# Googleビジネスプロフィール投稿の自動化に向けた準備

対象: ゆうき整骨院のGoogleビジネスプロフィール ／ 費用: 無料
現状: **手動投稿モードで運用中**（ジョブ画面から本文コピー→business.google.com に貼り付け）

## なぜ今すぐ自動化できないか

Googleの投稿API（Business Profile API）は**利用申請の承認制**で、
申請から承認まで通常1〜2週間かかる。承認が下りるまでは手動投稿モードが正式な運用。

## 申請手順（15分・承認待ち1〜2週間）

1. https://console.cloud.google.com/ にGoogleアカウント（ビジネスプロフィールのオーナー権限があるもの）でログイン
2. 新しいプロジェクトを作成（名前: `yuuki-marketing` など）
3. https://developers.google.com/my-business/content/prereqs の「APIへのアクセスをリクエスト」から申請フォームを開く
4. フォームに入力:
   - プロジェクトID: 手順2で作ったもの
   - ビジネスの説明: 整骨院1院の自院プロフィールへの投稿自動化（自社利用）
   - 想定利用: 週数回の投稿作成（localPosts）
5. 送信後、承認メールを待つ（1〜2週間）

## 承認が下りたら

アプリ側の実装（OAuth接続→localPosts投稿）はこちらで対応するので、
承認メールが届いたら知らせてください。それまでは手動投稿モードのままで問題ない。

## 補足

- 承認前でも投稿・口コミ返信は business.google.com から今まで通り手動でできる
- 口コミ返信のAI下書きはマーケティングハブの「口コミ返信」画面で承認前から使える
