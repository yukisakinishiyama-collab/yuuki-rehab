# お子さんのケガからの「復帰」、焦っていませんか？ ―― 保護者の方へ知ってほしい考え方

お子さんがスポーツ中にケガをしたとき、親としては「早く元通りにプレーさせてあげたい」と思う一方で、「無理をさせて悪化しないか」という不安も抱えるものです。

「試合が近いから」「レギュラー争いから外れてしまう」「本人が早く復帰したがっている」――こうした気持ちはとてもよく分かります。同時に、下関で多くのスポーツ外傷（運動中のケガ）を診てきた立場から、「復帰の焦り」がかえって回り道になるケースを何度も見てきました。

この記事では、お子さんの復帰について、保護者の方に知っておいてほしい考え方を整理してお伝えします。

## なぜ「痛みが消えた＝復帰OK」ではないのか

多くの方が、「痛みがなくなったから、もう大丈夫」と考えます。ですが、痛みの消失と身体機能の回復は、必ずしも一致しません。

たとえば足首の捻挫（靭帯を痛めるケガ）では、痛みが引いても、関節を支える筋力やバランス感覚（固有感覚といって、関節の位置を感じ取る能力）が十分に戻っていないことがよくあります。この状態で急に激しい運動に戻ると、同じ場所を再び痛めやすくなります。

つまり、復帰の判断には「痛みがないか」だけでなく、

- 左右差なく力が入るか
- 元の動作（走る・跳ぶ・切り返す）がスムーズにできるか
- 片脚立ちなどのバランスが保てるか

といった、複数の視点からの確認が必要になります。成長期のお子さんは骨や関節が発達の途中にあるため、大人以上に慎重な評価が求められます。

## 焦って復帰させると、どんなリスクがあるのか

十分に回復しないまま復帰することには、いくつかのリスクがあります。

**再発しやすくなる**
一度痛めた部位は、機能が戻る前に負荷をかけると同じケガを繰り返しやすくなります。捻挫を繰り返すうちに関節が緩くなり、慢性的な不安定さにつながることもあります。

**別の部位をかばって痛める**
痛みや違和感をかばう動きが癖になると、そのぶん他の部位に負担が集中します。「膝をかばっていたら腰を痛めた」といった二次的なケガは珍しくありません。

**プレーへの不安が残る**
身体が回復しても、「また痛めるのでは」という不安が残ると、思い切ったプレーができなくなります。心理的な準備も、復帰の大切な要素です。

こうしたリスクを避けるためにも、復帰は「時間が経ったから」ではなく、「段階を踏んで確認しながら」進めることが大切です。

## ご家庭でできること・見守りのポイント

専門家のサポートを受けることが前提ですが、ご家庭でも意識できることがあります。

**急性期（ケガ直後）は安静と冷却を基本に**
腫れや強い痛みがある時期は、無理に動かさないことが大切です。患部を冷やし、可能なら少し高い位置に保つと腫れを抑えやすくなります。

**「本人の申告」だけに頼らない**
お子さん、特に試合に出たい気持ちが強い子ほど、「もう痛くない」と実際より軽く伝えることがあります。歩き方や動きに左右差がないか、表情に無理がないか、保護者の目でも見守ってあげてください。

**段階的に負荷を上げる意識を持つ**
いきなり全力のプレーに戻すのではなく、軽い運動から少しずつ強度を上げていく。この「段階的復帰」の考え方をご家庭でも共有しておくと、焦りを防ぎやすくなります。

**痛みが出たら一歩戻る**
負荷を上げた後に痛みや腫れがぶり返した場合は、無理せず一段階前に戻す。この柔軟さが、結果的に早い復帰につながります。

## こんなサインは、早めに医療機関へ

以下のような症状が見られる場合は、自己判断せず、まず整形外科などの医療機関を受診してください。

- 強い腫れや変形があり、体重をかけられない
- 患部を押すと骨に沿って強い痛みがある
- しびれや感覚の異常、力が入らない
- 頭部を打った後の頭痛・吐き気・意識のもうろう
- 数日たっても痛みや腫れがまったく引かない

これらは骨折や重度の損傷、あるいは緊急の対応が必要なサインである可能性があります。「たいしたことないだろう」と様子を見てしまうと、回復を遅らせることにもなりかねません。

## 復帰は「安全に、確実に」を一緒に考えます

医療機関で骨などに問題がないと確認できた後、あるいはリハビリの段階に入ってからは、私たちのような施術院が力になれる場面が増えてきます。

「いつ頃から練習に戻せそうか」「何に気をつければいいか」といったご相談だけでも構いません。保護者の方の不安に寄り添いながら、一緒に考えていきます。
