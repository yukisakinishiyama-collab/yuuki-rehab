# 1ヶ月のまとめ:「段階的復帰」という考え方 — 焦らず、確実に戻るために

「もう痛みはないから、明日から前と同じように運動していいですか？」

ケガや手術のあと、当院でこうした質問を受けることが本当に多くあります。早く元の生活やスポーツに戻りたい——その気持ちは、とてもよく分かります。でも、痛みが消えたことと、身体がもとの動きに耐えられる状態に戻ったことは、実は同じではありません。

この1ヶ月、当院では「段階的復帰(だんかいてきふっき)」という考え方を大切にした施術やアドバイスをお伝えしてきました。今回はそのまとめとして、なぜ「一気に戻す」のではなく「段階を踏んで戻す」ことが大切なのかを、あらためて整理してお話しします。

## なぜ「痛みが消えた=復帰OK」ではないのか

痛みは、身体が発する最初のサインのひとつですが、回復の“ゴール”を知らせてくれるものではありません。

たとえば足首の捻挫(ねんざ)を例に考えてみましょう。腫れが引いて日常生活で痛みを感じなくなっても、その内部では次のような課題が残っていることがあります。

- **筋力の低下**：安静にしていた期間に、その部位を支える筋肉は思った以上に落ちています
- **バランス感覚(固有受容感覚)の低下**：関節の位置や傾きを感じ取る能力が鈍り、とっさの動きに対応しにくくなっています
- **動作パターンの乱れ**：かばう動きが癖になり、別の部位に負担が移っていることもあります

つまり、痛みという「表面のサイン」が消えても、身体の機能そのものはまだ回復の途中、というケースは珍しくないのです。ここを見極めるために、当院では痛みだけでなく、可動域・筋力・動作の質などを評価しながら復帰の段階を判断していきます。

## 急な全力復帰が招くリスク

「もう大丈夫」と自己判断で一気に元の運動量に戻してしまうと、いくつかのリスクが生じます。

最も多いのが**再発**です。まだ十分に機能が戻っていない部位に、以前と同じ負荷を突然かければ、同じ場所を痛めてしまう可能性が高まります。捻挫を繰り返すうちに関節が緩くなり、慢性的な不安定さにつながることもあります。

もうひとつが**別の部位のケガ**です。かばう動きが残ったまま無理をすると、反対側の脚や腰、膝など、思わぬ場所に負担が集中してしまうことがあります。「良くなったはずの足首は平気なのに、今度は膝を痛めた」という流れは、決して他人事ではありません。

術後のリハビリでも同じことが言えます。手術で修復した組織が十分な強度を取り戻す前に負荷をかけすぎると、回復の妨げになることがあります。焦りは、結果として復帰までの時間を長引かせてしまうこともあるのです。

## 段階的復帰の基本的な進め方

段階的復帰とは、身体の状態を確認しながら、負荷を少しずつ・順序立てて上げていく考え方です。おおまかには、次のような段階を踏んでいきます。

1. **痛みと炎症の管理**：まずは腫れや痛みを落ち着かせ、日常動作を無理なく行える状態を目指します
2. **可動域の回復**：関節が本来動くべき範囲を、少しずつ取り戻していきます
3. **筋力とバランスの再獲得**：落ちてしまった筋力や、関節を支える感覚を鍛え直します
4. **動作・スポーツ動作の練習**：走る・跳ぶ・切り返すなど、実際の場面に近い動きへ段階的に近づけます
5. **競技・生活への完全復帰**：負荷に耐えられることを確認したうえで、もとの活動へ戻ります

大切なのは、**各段階を「クリアしてから」次へ進む**という順序です。前の段階で痛みや不安定さが出るようなら、一段戻って調整します。これは後退ではなく、確実に前へ進むための調整だと考えてください。

### ご自身でできること

専門家のサポートを受けながらも、日々のセルフケアはとても大切です。

- **痛みを「ものさし」にする**：運動中や翌日に痛みが強く出るなら、負荷が今の身体に合っていないサインです
- **左右差を意識する**：ケガをした側だけ極端に細くなっていないか、鏡や日常動作で確認してみましょう
- **睡眠と栄養を整える**：組織の回復には、しっかり休むことと十分な栄養も欠かせません
- **記録をつける**：「今日はここまでできた」を書き留めると、復帰の目安がつかみやすくなります

## こんなときは医療機関へ

段階的復帰を進めるなかで、次のようなサインがある場合は、自己判断で運動を続けず、整形外科などの医療機関を受診してください。

- 安静にしていても強い痛みが続く、夜間に痛みで目が覚める
- 関節が大きく腫れている、熱を持っている、変形しているように見える
- しびれや力が入らない感覚がある
- 体重をかけられない、まったく動かせない

これらは、自己ケアの範囲を超えた対応が必要なサインの可能性があります。「様子を見よう」と我慢しすぎず、まずは専門的な検査を受けることが安心につながります。

## おわりに

段階的復帰は、遠回りに見えて、実はもっとも堅実に元の生活へ戻るための道すじです。焦らず、今の身体の状態に合わせて一歩ずつ——この考え方が、再発を防ぎ、長く健康に動ける身体づくりにつながります。
