# 通院頻度、何を基準に決める? ―「毎日来て」と言われないために知っておきたいこと

「週に何回通えばいいんだろう?」
整骨院や整体に通い始めたとき、多くの方がこの疑問にぶつかります。

「毎日来てください」と言われて負担に感じたり、逆に「もう来なくていいのかな?」と自己判断でやめてしまったり。通院ペースは、痛みの改善にも、時間やお金の使い方にも直結する大切なテーマです。

今回は、通院頻度を「なんとなく」ではなく「根拠を持って」考えるための基準を、下関市彦島のゆうき整骨院がお伝えします。

## そもそも通院頻度はなぜ変わるのか ―「症状の段階」がカギ

通院頻度に一律の正解はありません。なぜなら、体の状態は人によっても、時期によっても違うからです。

大きく分けると、症状には次のような段階があります。

- **急性期**：ケガや痛みが出た直後。炎症（腫れ・熱・強い痛み）が強い時期
- **回復期**：炎症が落ち着き、組織が修復されていく時期
- **メンテナンス期**：痛みがほぼ治まり、再発予防や機能改善に取り組む時期

一般的に、**急性期は組織の状態が変わりやすいため、こまめに経過を確認したほうがよいことが多い**です。捻挫直後などは、その日の腫れ具合や動かせる範囲を評価しながら施術を調整する必要があります。

一方、回復期・メンテナンス期に入ると、体が良い状態を保てる期間が延びていくため、間隔を空けていくのが自然な流れです。

つまり通院頻度とは、「決められたノルマ」ではなく、**今の症状がどの段階にあるかを評価した結果として決まるもの**なのです。

## 「とにかく毎日」「一律で週3回」に注意したい理由

ここで少し立ち止まって考えたいのが、症状や経過にかかわらず一律の頻度を勧められるケースです。

もちろん、状態によっては短期間の集中的な施術が適切な場合もあります。問題は、**「なぜその頻度なのか」の説明がないまま回数だけが決まっていく**ことです。

通院頻度が体の状態と合っていないと、こんなすれ違いが起こりがちです。

- 本当はこまめなケアが必要な時期なのに、間隔が空きすぎて回復のタイミングを逃す
- すでに落ち着いているのに通い続け、時間や費用の負担だけが増える
- 自己判断で急にやめてしまい、再発したりクセになったりする

特に見落とされやすいのが**再発予防の視点**です。痛みが消えた＝完全に元通り、とは限りません。ケガの背景には、体の使い方のクセや筋力・柔軟性の低下といった「原因」が隠れていることがあります。ここを評価せずに通院をやめると、同じ痛みを繰り返す一因になることもあります。

大切なのは、頻度そのものより「**その頻度に納得できる理由があるか**」です。

## 自分で通院ペースを見直すためのチェックポイント

通院を続けるなかで、「今のペースが合っているかな?」と感じたら、次の視点で振り返ってみてください。

**1. 症状が変化しているか**
痛みの強さ、動かせる範囲、日常動作のしやすさ。前回と比べて少しずつでも変化があるかどうかは、大切な手がかりです。

**2. 施術の目的が共有できているか**
「今は炎症を抑える時期」「これからは動きを取り戻す時期」など、今どの段階で何を目指しているのかを、施術者と一緒に理解できていると安心です。

**3. 自宅でのセルフケアが提案されているか**
通院は回復のサポートであって、すべてではありません。ストレッチや姿勢の工夫など、**通っていない日をどう過ごすか**の提案があると、頻度も自然と調整しやすくなります。

**4. 生活・仕事・スポーツの予定と両立できているか**
無理な通院はストレスになり、かえって続きません。復帰したい試合や仕事の状況を伝えることで、現実的なペースを一緒に考えられます。

疑問があれば、「この頻度で通う理由を教えてください」と遠慮なく質問して大丈夫です。誠実な施術者であれば、根拠を持って説明してくれるはずです。

## こんなサインがあれば、まず医療機関へ

セルフチェックの前に、必ず知っておいていただきたいことがあります。次のような症状がある場合は、通院頻度を考える以前に、**整形外科などの医療機関を早めに受診してください**。

- 強い腫れや変形があり、体重をかけられない・動かせない
- 手足のしびれや力の入りにくさが続く
- 安静にしていても強い痛みがある、夜間に痛みで眠れない
- 発熱を伴う痛み、痛みが日に日に強くなる
- ケガの後、時間が経っても症状がまったく改善しない

これらは、レントゲンや精密検査が必要な状態（骨折や神経の障害など）が隠れているサインかもしれません。自己判断や施術で様子を見ず、まずは医療機関での診断を優先することが、遠回りに見えて一番の近道です。

## 通院頻度は「一緒に決めていく」もの

通院頻度は、施術者が一方的に決めるものでも、患者さんが我慢して合わせるものでもありません。**今の状態を評価し、目標を共有しながら、一緒に調整していくもの**です。
